Hexo

Jade barebone theme

Intro

27 December 2013 in Articles 

Barebone theme for Hexo implemented with Jade and Less. The current Hexo theme list does not have anything other than ejs themes. I made this as a starting point for anyone who is interested in theming using Jade. Check the documentation to learn more about Hexo.
